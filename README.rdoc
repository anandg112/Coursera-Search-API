== README

This Ruby app uses Coursera API to search for courses based on search query. The default is jhu. To change the search query, please add the following at the end of the URL <tt>?looking_for=search_term</tt> where search_term corresponds to the keyword you are looking for.

* System dependencies - Please install HTTParty gem.

* The app is deployed on Heroku at - https://coursera-search-anand.herokuapp.com/
